Les jeux plus joues

Voici ici les jeux préférés des joueurs francais

  1. Minecraft

    2. Bienvenue dans ce magnifique jeu cubique. Minecraft est un jeu avec différents mode de jeu. Vous pouvez choisir de jouer en survie, en créatif, en Aventure ou en Hardcore.

  2. Fortnite

    3. Dépéchez vous ! Le bus risque de partir sans vous ! Fortnite est un magnifique jeu au style cartoon dont le but est de se battre avant d'être le seule survivant de l'île. Le jeu comporte cette Battle Royale mais également nombre de mini-jeux fait par la communauté.

  3. Fall Guys

    4. Trébuchez vers la gloire. Fall Guys est un jeu festif gratuit, multiplateformes, massivement multijoueur de battle royale dans lequel vous et vos adversaires vous affrontez manche après manche dans une course d'obstacles chaotique jusqu'à déterminer un vainqueur !

  4. League of Legends
  5. PUBG
